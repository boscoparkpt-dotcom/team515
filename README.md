# Team515 — NY/NJ 한인 철인클럽 앱

설치형 웹앱(PWA) · 레이스 캘린더 · 정기 훈련 일정 · 운영 규정.

## 구조
- `index.html` — 앱 본체 (홈/레이스/훈련/클럽 4탭, PWA)
- `data.js` — 레이스·일정·멤버 데이터 (+ 구글시트 연동 URL 칸)
- `manifest.json`, `sw.js` — PWA(홈 화면 설치) + 오프라인/자동 최신
- `icons/` — 앱 아이콘
- `races-template.csv`, `구글시트-연동방법.md` — 시트로 셀프 관리
- `입장규정-요약.md` — 오픈채팅 공지용 규정 요약

## 업데이트
`data.js`만 고쳐서 배포하거나, 구글시트를 연결하면 운영진이 시트만 수정해도 자동 반영됩니다.
