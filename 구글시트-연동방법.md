# 레이스 캘린더 — 구글시트로 셀프 관리하기

목표: **운영진 누구나 시트에 줄 하나 추가하면 앱에 자동 반영.** 코드·재배포·보스코 불필요.

## 1회 셋업 (5분, 보스코가 1번만)

1. **새 구글시트 생성** → `races-template.csv`를 import (파일 → 가져오기 → 업로드)
   - 첫 줄(헤더)은 `date, name, dist, place, members` 그대로 유지
2. 시트 탭 이름은 아무거나 OK (예: `races`)
3. **웹에 게시**: 파일 → 공유 → **웹에 게시** → "쉼표로 구분된 값(.csv)" 선택 → 게시
4. 나온 **CSV 링크**를 복사 → `data.js`의 `SHEET_CSV_URL: ""` 안에 붙여넣기
   - 예: `SHEET_CSV_URL: "https://docs.google.com/spreadsheets/d/e/2PACX-.../pub?gid=0&single=true&output=csv",`
5. 재배포 1회 (또는 보스코한테 "시트 링크 이거야" → 내가 넣고 배포)

→ 이후로는 **시트만 고치면 끝.** 앱 다시 안 만져도 됨.

## 입력 규칙

| 컬럼 | 뜻 | 예시 |
|------|----|------|
| `date` | 날짜 (YYYY-MM-DD). 미정이면 월까지 + `-XX` | `2026-07-19` / `2026-09-XX` |
| `name` | 대회명 | `IRONMAN 70.3 Jones Beach` |
| `dist` | 거리: `sprint` `oly` `half` `full` `etc` | `half` |
| `place` | 장소 (쉼표 있으면 따옴표로) | `"Jones Beach, NY"` |
| `members` | 참가자 (쉼표로 여러 명) | `보스코, 레이, 쌤` |

- 줄 추가 = 레이스 추가 / 줄 삭제 = 제거 / 셀 수정 = 수정
- members 비워두면 앱에 "참가자 모집 중"으로 표시
- 운영진에게 시트 **편집 권한** 공유하면 분담 관리 가능

## 안전장치
시트가 비었거나·삭제됐거나·인터넷이 끊겨도, 앱은 `data.js`에 박아둔 기본 레이스로 자동 폴백해서 **절대 빈 화면이 안 됨.**
